# Dream Team Real Estate UI

Premium real estate brokerage landing page for a Mongolian property sales company.

## Included

- Next.js App Router scaffold
- Tailwind CSS theme
- Premium real estate hero section
- Featured property cards
- Service and value proposition section
- Property valuation lead form
- Responsive mobile and desktop UI

## Run locally

```bash
npm install
npm run dev
```

## Design direction

- Premium, trustworthy, high-value brokerage feel
- Deep green, cream, and gold palette
- Lead-generation first UX
- Clear CTAs for valuation and advisor contact
